
In this github repository, you can find the files that belong to the article:
Kuiper, R. (2021). AIC-type Theory-Based Model Selection for Structural Equation Models. 
Structural Equation Modeling. https://doi.org/10.1080/10705511.2020.1836967


In the meantime, the R package restriktor has been changed, meaning that the code to run the examples changed.
The files based on the newest restriktor version can be found on:
https://github.com/rebeccakuiper/Tutorials/tree/main/GORICA%20in%20SEM



How to download the files:

1. Click on green button called Code.
2. Download zip (last option in list).
3. Unzip it on your machine (that folder is now your working directory).
